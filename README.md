# DSA Visualizer

Interactive data structures and algorithms visualizer built from the developer
guidelines in `dsa-developer-guidelines.docx`.

## Step-by-step roadmap

1. Sorting foundation: Next.js app shell, generator-based sorting engine,
   Zustand stores, Framer Motion bar canvas, controls, array input, and synced
   code panel.
2. Trees: BST insert/search and traversal snapshots.
3. Graphs: BFS, DFS, and Dijkstra with animated SVG nodes and edges.
4. Search: linear and binary search pages.
5. Portfolio readiness: responsive polish, Lighthouse pass, CI, and Vercel
   deployment.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality checks

```bash
npm run type-check
npm run test
npm run build
```
